

安禄山、史思明的灭亡，归根结底，是他们自取灭亡。

四大贼首互相残杀，最终同归于尽，这并非是大唐军队靠俘虏或斩首换来的胜利。

先前收复长安、凤翔两京，后来收复东都洛阳，都是趁着敌军内部衰败，并借助了回纥的兵力。李光弼和郭子仪，也是顺应了时势才建立了大功，并非全靠血战拼杀得来的殊勋。

如果单以战功而论，李光弼智勇双全，克敌制胜的功绩要比郭子仪多得多；反观郭子仪，清渠一战失利，相州一战溃败，在战绩上其实是短绌的。

然而，若要问谁才是大唐社稷的柱石之臣，谁能让天下的安危系于一身，后世的评价却没有任何异议，一致推崇郭子仪。

这是为什么？

因为承担天下重任的人，最伟大的品质，莫过于平心静气地顺应事物的顺逆变化，而不夹杂个人的私意，不自以为是地居功自傲。这种境界，只有古时那些真正懂得大道的人才能做到。

《诗经》赞美周公的德行，说他心志安定，无论身处何地都能安然自处。

那些总想把天下大权揽在自己手里的人，内心其实充满了危机感；那些怀疑天下人、不敢自己担当责任的人，内心其实充满了偏颇。心藏在身体里，却是掩盖不住的。正因为它无法掩盖，所以天下人都看在眼里。

你想以此得利还是心生厌倦，你是心存疑虑还是彼此信任，天下人都会以同样的态度，分毫不差地回报给你。

郭子仪（汾阳王）出身贫寒，当年靠着体力起家，但他却暗合了这种大道。他这种得天独厚的气质，是夏商周三代以后，无人能比的。

历史上，有些人能担当重任，却不懂得谦让，这就是所谓的豪杰之士，像韩信、马援便是如此；有些人懂得谦让，却不能担当重任，这就是所谓的明哲保身之徒，像张良、李泌便是如此。

而郭子仪，对于地位的升降、权力的去留、皇帝的疑信、谗言的起灭，乃至于功业的成与不成，只要事情来了，他就接受；既然接受了，他就担当；在这期间，绝不夹杂任何个人的私心杂念。

他的心态平静到了极点，以至于天下人都猜不透他的所作所为。

山有起伏，人们才能测算出峰峦的高低；水有险滩，人们才能测算出波浪的回旋。如果一片平坦浩荡，无高无下，无曲无奇，旁人又从何测度呢？

既然天下人都看在眼里，却始终无法测度他的深浅，这就太伟大了！这就是“平情”——心态平和的作用。四海之内的万事万物，都在他的气度包容之中；无论贤良还是不肖，千万种不同的人心，最终都归于他的节制范围之内。

相州大军溃败后，郭子仪威名受损。加上宦官鱼朝恩进谗言，肃宗便夺了他的兵权，转授给李光弼。

在此后的数年里，李光弼作为元帅，手握重兵，在中原全力拼杀，声势似乎要凌驾于郭子仪之上。

然而结果呢？许叔冀在汴州叛变，刘展在江淮造反，段子璋在梓州作乱，楚州杀了李藏用，河东杀了邓景山，行营里杀了李国真、荔非元礼。内乱蜂起，此起彼伏。

直到宝应元年，郭子仪受封王爵，重新主管各道行营，天下这才安定下来。内部既然安宁，外部的敌人自然不攻自破。史朝义就像锅里的游鱼，惶惶不可终日，根本不需要再经过什么血战，便自行灭亡了。

呜呼！这种局面，哪里是李光弼的智勇所能企及的？这哪里是汉魏以后的那些将相大臣，能够从天下人那里得到的拥戴？

当年，董卓不足以灭亡汉朝，真正灭亡汉朝的，是关东的诸侯；桓玄不足以灭亡晋朝，真正灭亡晋朝的，是北府兵的内乱；黄巢不足以灭亡唐朝，真正灭亡唐朝的，是汴州和晋州的藩镇。

同样的道理，安禄山和史思明，其实并非唐朝最大的忧患。那些乘机蜂拥而起的各方势力，才是真正的威胁，谁知道天下这头“鹿”最终会死在谁手里？

郭子仪一出山，天下的战火就熄灭了。他重树威信，不过是斩杀了王元振等四十多人而已，天下便再也没人敢作乱。

正是因为他能平心静气地看待权势的去留，能做的就做，不能做的就止。这种坦然无我、顺应大道的境界，别人用世俗的心思去揣测他，根本摸不着头脑。又有谁知道，这其实就是顺应本心、顺应万物，自然而然地接受荣辱的结果呢？

仆固怀恩本是个容易作乱的人。当时有人想驱逐李光弼，仆固怀恩却说：“邺城溃败，是因为郭公先撤退了，朝廷要责罚主帅，所以才罢免了他的兵权。”

他竟然主动引咎归己，来安抚众心，这话听起来多么像君子之言！如果不是郭公这种安土敦仁、不舍弃任何人的气度，深深地浸润了众人的心，像仆固怀恩这样的人，怎么可能说出这种话来？

做臣子的道义，忧国如家，这是本性的节操；社稷的重任在肩不可推辞，这是道义的责任。

那些虽然忠贞却过于执着的人，急切地谋求成功，一旦形势受阻、力量不济，心中必然产生不平之气。这种意气一旦在心中萌发，必然会表现在外。于是天下人便会争相追逐功名，而忘记了忠顺的本分。奸人趁机利用这一点，祸乱也就因此而起了。

只有像郭子仪这样，把立功报国的心情，平淡地担当起来，心中没有任何强求成功的执念，用这种冰消雪融、风轻云淡的胸怀面对天下，凡是可以做的无不尽力去做。那么，即使有桀骜不驯、图谋不轨之徒，也会感到气势受挫、心灰意冷，而不敢妄动。

这是不言之言，无功之功。回纥人称他为“大人”，确实啊，他真是一位真正的“大人”。

李光弼的忠勇，虽然不在郭公之下，但天下人却没能得到他的庇护。这两位将相放在一起衡量，气度格局的高下，已经显而易见、清清楚楚了。
