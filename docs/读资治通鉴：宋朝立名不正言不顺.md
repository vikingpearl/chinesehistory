
所谓“五代”，这五个被后世史家合称为朝代的政权，其实根本不配拥有国号。

朱温，不过是个窃国的大盗，其性质与安禄山之流并无二致；李存勖、石敬瑭、刘知远，不过是沙陀族三个部落的夷狄首领；郭威则是靠着趁火打劫、篡位夺权起的家，仅仅是为了有个名分才妄称帝王。即使是后周世宗柴荣，虽然他最初没有参与篡逆的阴谋，是以养子的身份继位，并且确实有着平定天下的志向，因此我们尊称他为周主，认为他胜过那些夷狄与强盗，但归根结底，这一群人都不足以担当“天子”的神圣名号。

把这段时期称为“五代”，那纯粹是宋朝人的说法。这哪里配得上称“代”呢？

所谓“代”，意味着统绪的承接，道义的延续，以及制度文明的更替。商汤取代夏桀，武王讨伐商纣，那是真正的革命，他们统一了天下，继承前代的礼乐并加以改良，这才能称为“三代”。反观朱温、李存勖、石敬瑭、刘知远、郭威这些卑琐之徒，仅仅是窃据了唐朝的京师，就能算作是一“代”吗？郭威虽然不是夷狄也不是盗匪，稍微接近一点正统，但他不过是一个受过黥刑的士兵乍然兴起，既无显赫功业，又是趁着孤儿寡母之际夺取政权，甚至不如南北朝时期平定战乱的陈霸先，相比之下郭威简直如奴隶一般。至于朱温，纯粹是强盗；李存勖等人，不过是像犬羊一般的沙陀部落酋长。如果朱温这种强盗可以取代唐朝，那么侯景之乱的侯景也能取代梁朝，山东的一介贼寇李全也能取代宋朝了；如果沙陀三部可以成为中华之主，那么五胡乱华时的匈奴刘聪、羯族石虎也能取代晋朝了。

况且，这五个人，何曾真正拥有过天下？

在朱温称帝时，晋王李克用已经与他分庭抗礼，李茂貞、刘仁恭、王镕、罗绍威这些人也都拥兵自重，互不服气。其他的如南方的杨行密、徐知诰、王建、孟知祥、钱镠、马殷、刘隐、王潮、高季兴等人，先后割据一方，自己称帝称王，分土而治。虽然他们有的借用了中原的年号，但实际上何曾穿戴汉家衣冠来朝拜？何曾进行过春秋祭祀？何曾有一日奔走效命于汴梁和洛阳？

如果说，仅仅因为占据了汴梁、洛阳这些王者之地，就能算作正统，那么舜帝和大禹接受禅让时，并没有沿用尧帝的陶唐旧居；商汤和周武王革命时，也没有定都在夏商的旧都。反之，前秦的苻健、后秦的姚兴、北魏的拓跋宏，他们都曾占据过汉晋的故宫，难道我们就要承认他们是正统吗？如果仅仅依据张文蔚这帮文人撰写的禅让诏书，就承认朱温可以取代唐朝，那简直是奖赏天下的叛逆，蔑视国家的神器！

要想成为一个改朝换代的天下共主，不仅要以君主自居，更要让天下人真心视你为君主。即便做不到尽善尽美，至少要有治理天下的志向。秦朝、隋朝虽然无道，但至少还创立法律制度，试图以此约束天下，虽然悖乱，但在因时救弊方面尚有可取之处。哪怕是狂妄愚蠢的王莽，他在篡位后也是早起晚睡，一心一意为了生民在规划疆域。

可是，现在的石敬瑭、刘知远之流，只顾着苟且偷生、窃取一时的尊位，根本不值一驳。即便是李克用父子归附鞑靼以后，朱温统领宣武军以来，他们觊觎皇位已经不是一天两天了，但君臣之间抵掌密谋的时候，可曾有一念想过百姓的死活？可曾有过一点立国的宏图规模？他们竭尽智力所谋划的，只有互相撕咬、毒害生灵、争抢地盘，以满足私欲。他们手下的谋士，如敬翔、李振、周德威、张宪之流，也不过是含着毒液、伸着利爪互相抓挠的野兽。

所以，朱温一篡唐，李存勖一灭后梁，便开始了淫乱暴虐，根本不给百姓留活路。他们胁迫臣民，只为了满足无底洞般的贪欲；他们的制度设施，全盘照搬唐朝末年的弊政，再加上藩镇特有的疯狂胡为。这与刘守光、孟知祥、刘龑、王延政、马希萼、董昌这些人的志向有什么区别？作恶程度也是半斤八两。当时天下纷乱，人人称帝称王，为什么要单单挑出这五个人，私心地称他们为“代”呢？这五个人最初就没有君临天下的志向，天下人也没有拥戴他们的心意，就像燎原的野火，烧起来快，灭得也快，成了一堆灰烬，这哪里配得上叫“代”！

如果万不得已，非要在这个时期推举一个君主，那么南方的杨行密、徐温、王建、李昪、钱镠、王潮这些人甚至更胜一筹！因为他们尚存一丝爱护百姓之心，百姓也能依附他们求得暂时的安宁。

回顾历史，周朝自威烈王以后，七国争雄，诸侯画地为牢等待灭亡，虽然周赧王最终向秦国献地投降，但天下后世并不认为秦国立刻就取代了周朝，而是称这段时期为“战国”。那么，从唐朝天祐年间灭亡，到宋朝建隆年间建立，这段时期也应当称之为“战国”，这才是恰当的！为什么要从那些偏安一隅、迅速灭亡的强盗和夷狄中，硬推崇出一个所谓的“天下共主”呢？

中国不可一日无君，就像人不可一日无父。但是，如果孤儿无法持家，那就是失去了父亲，绝不能因为旁边有个强横的恶仆或者强邻，就认贼作父，称他为父亲。如果这样，世上就有了没有父亲的儿子，也有了没有君主的臣民。这是人伦的常理，天道的昭示，绝不可混淆视听。

宋朝得天下的手段，本就不正，它是从后周孤儿寡母手中夺来的。为了证明自己权力的来源是合法的，宋朝不得不推崇柴氏的后周，既然承认了后周，就顺带承认了之前的四个政权，甚至承认了朱温取代唐朝的合法性，于是“五代”这个名号就确立了。

名分是不可假借的，天下分裂就是分裂，夷狄盗贼窃国就是窃国，绝不可纵容。把这些人的国号拿来，统称为“五代”，这不过是为了掩饰宋朝人自己得位不正的尴尬罢了！
